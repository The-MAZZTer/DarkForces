Release Notes

Browsers verified to work: Google Chrome, Microsoft Edge, Mozilla Firefox

You can add arguments to the query string which will be parsed as command line parameters. Example:

?tool=LevelExplorer

On case sensitive systems (and WebAssembly), Dark Forces files may not be detected if the file names are not all upper case .

The ESC key may not function in the Level Explorer to open the menu. Use the ` key instead.

All files generated with this tool are only saved in memory and will be lost if not downloaded through the browser before closing this tool.

Use the download button in the lower left to download files you want to save.

3DO/VUE files may fail to preview in the Resource Editor after running for some time. Reload the app to fix.
